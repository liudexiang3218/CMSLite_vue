# CMSLite Vue

#### 项目介绍
CMSLite是一套极简的创建内容管理系统，可用于创建企业网站，个人blog，目前只提供一套模板。
CMSLite是本人的一个前端练手项目的后端API加网站展示，同时本人也是在这个框架基础上搭建了几套企业网站。
##### 项目功能
1. 用户管理
2. 文章管理
3. 栏目管理
4. 产品管理
5. 产品分类
6. 网站基础设置
7. 产品组件
8. 留言管理
9. banner管理
10. 区块管理

#### 安装教程

1. 下载项目 git clone https://github.com/liudexiang3218/CMSLite_vue.git
2. cd CMSLite_vue
3. npm install
4. npm run dev
5. http://127.0.0.1:9527
